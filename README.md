# NMTI

**牛马型人格测评 · 职场搞笑版**

**创意灵感来自 [SBTI](https://sbti.unun.dev)**——网络上对 MBTI 的恶搞向趣味测评风潮（多题、多标签、强玩梗、纯娱乐）。**NMTI 为独立企划**：沿用四字母形态做戏仿，把场景落到当代职场「牛马」自嘲，与 SBTI 无官方关联。通过职场场景选择题生成「NMTI」类型名与判词，**并非科学心理测评**，仅供自嘲与社交分享。

---

## 项目梗概

- **玩法**：12 道与领导、流程相关的场景题；四轴计分拼出四字母（如 `ESTJ`），对应 **16 种**预设人格文案。
- **标准版**：题库随机抽题、题目顺序打乱；选项带 **A / B / C / D** 前缀，顺序与配置一致。
- **高血压版**：首次完成标准版并看到结果后**惊喜解锁**（首页不剧透）；题库更典、讽刺更强，结果页含按类型区分的「血压评语」。
- **其它**：答题支持「上一题」；页头「清理缓存」可清除本机解锁记录并重置到欢迎页（便于彻底重测）。

---

## 做这个的意图

用轻量网页做**自嘲与传播**：熟人之间转发类型名与判词；解锁设计尽量保留「第一次通关的惊喜感」。  
**再次说明**：全站内容为**娱乐向**，不构成任何职业或心理健康建议。

## 技术栈（与当前实现一致）

| 项目 | 说明 |
|------|------|
| 运行时 | 纯静态：**HTML5**、**CSS3**、**JavaScript（ES 模块）** |
| 架构 | 极简 **EventBus** + **quizState** 状态机（欢迎 / 答题 / 结果） |
| 配置 | 题库、十六型文案、常量见 [`js/config.js`](js/config.js) |
| 构建 | **无**打包步骤；本地需 HTTP 访问（勿用 `file://` 直接打开） |

更完整的工程约定（含后续可选的 Canvas、Howler、Vite 等）见仓库内 [`TECH_STACK.md`](TECH_STACK.md)；**当前线上形态未使用**上述扩展项。

## 本地运行

在项目根目录任选其一：

```bash
python -m http.server 8080
```

或：

```bash
npx serve .
```

浏览器访问终端提示的本地地址（如 `http://127.0.0.1:8080`）。
